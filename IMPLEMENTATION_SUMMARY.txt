PASO 1: Agregar dependencia (5 min)
──────────────────────────────────
$ flutter pub add flutter_dotenv

PASO 2: Firebase Console Setup (15 min)
───────────────────────────────────────
1. Crear proyecto: https://console.firebase.google.com
2. Agregar app iOS
3. Agregar app Android
4. Descargar google-services.json → android/app/
5. Descargar GoogleService-Info.plist → ios/Runner/

PASO 3: Variables de Entorno (5 min)
─────────────────────────────────────
Crear file .env en raíz:

FIREBASE_API_KEY=tu_api_key_aqui
FIREBASE_AUTH_DOMAIN=feeltrip-app.firebaseapp.com
FIREBASE_PROJECT_ID=feeltrip-app
FIREBASE_STORAGE_BUCKET=feeltrip-app.appspot.com
FIREBASE_MESSAGING_SENDER_ID=tu_sender_id
FIREBASE_APP_ID=tu_app_id

PASO 4: Reglas Firestore (5 min)
────────────────────────────────
En Firebase Console → Firestore → Rules, pegar:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if true;
    }
  }
}

PASO 5: Correr la app (5 min)
──────────────────────────────
$ flutter pub get
$ flutter run
