# 🏔️ FeelTrip - Autonomous Travel Agent Ecosystem

FeelTrip no es solo una aplicación de viajes; es un **Sistema de Agentes Autónomos** diseñado para transformar el diario personal en una expedición planificada y validada. Construida con Flutter, Firebase y el poder agéntico de Google AI (Gemini).

---

## ⚡ Estado del Arte (Fase 5: Era Agéntica)
- **Scout Agent Engine:** Un núcleo de razonamiento autónomo que analiza tus emociones y utiliza herramientas reales (Clima, Vuelos, Calendario) para proponerte el próximo paso de tu vida.
- **Tool Calling Autónomo:** Integración con **Amadeus API** para vuelos reales y generadores de itinerarios inteligentes.
- **Arquitectura Moderna:** Migración completa a **Riverpod** para un estado reactivo, seguro y predecible.
- **Offline-First Nativo:** Persistencia local con **Isar & Hive** garantizando que tus crónicas se guarden en el corazón de la montaña, sin necesidad de red.
- **IoT / Wearables:** Sincronización bidireccional con **Wear OS** para telemetría y seguridad en ruta.

---

## 🛠️ Stack Tecnológico
- **UI:** Flutter (Material 3 + Cyber-Organic Aesthetics).
- **IA/Agentes:** Google Generative AI (Gemini 1.5 Flash) + Autonomous Tool Calling.
- **State Management:** Riverpod (Funcional & Reactivo).
- **Backend:** Firebase Suite (Auth, Firestore, Cloud Functions, Messaging).
- **Local DB:** Isar (Alta performance) + Hive (Configuración).
- **Vuelos:** Amadeus REST API + Google Flights Fallback.
- **Monetización:** RevenueCat (Suscripciones) + Mercado Pago (Beta).
- **Observabilidad:** Sentry + Firebase Crashlytics.

---

## ✨ Características Principales
- **🤖 Scout Agent:** El agente analiza tu diario, verifica el clima en el destino, busca vuelos reales en Amadeus y agenda tu calendario de forma proactiva.
- **📓 Experiential Diary:** Escribe tus sentimientos; la IA los convierte en datos accionables y crónicas vivenciales.
- **🌍 Social Feed:** Historias, Reels y comentarios en tiempo real sincronizados globalmente.
- **📊 B2B Dashboards:** Métricas avanzadas para agencias de viajes (Conversión, Leads, Impacto Emocional).
- **⌚ Wear Sync:** Tu expedición te acompaña en la muñeca, con alertas de POIs y estado físico.

## 🚀 Instalación Rápida
1. **Configurar Entorno:**
   ```bash
   flutter pub get
   ```
2. **Variables de Entorno (.env):**
   Crea un `.env` basado en `.env.example` con:
   - `GEMINI_API_KEY`
   - `AMADEUS_CLIENT_ID` / `AMADEUS_CLIENT_SECRET`
   - `SENTRY_DSN`
3. **Ejecutar:**
   ```bash
   flutter run
   ```
