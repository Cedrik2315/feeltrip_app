# 🎯 IR A FIRESTORE DATABASE

## EN FIREBASE CONSOLE

```
Panel IZQUIERDO:

Build (desplegable)
  ├─ Firestore Database  ← CLICK AQUÍ
  ├─ Realtime Database
  ├─ Storage
  ├─ Authentication
  └─ ...

Settings (engranaje)
  └─ Project settings (aquí estás ahora)
```

## PASOS:

1. En el panel izquierdo, busca **"Build"**
2. Haz click en **"Firestore Database"**
3. Espera a que cargue (1-2 segundos)
4. Deberías ver: **"Cloud Firestore"** con un botón azul **"+ Create Collection"**

---

## CUANDO ESTÉS EN FIRESTORE DATABASE:

- Verás Collections (vacío al inicio)
- O verás colecciones ya creadas (stories, agencies, etc)
- Ahí es donde crearemos los datos

---

**Haz click en "Firestore Database" y avísame cuando veas la pantalla con "+ Create Collection"**
