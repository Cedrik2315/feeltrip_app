# 📍 ESTÁS EN LA PÁGINA PRINCIPAL

## DESDE https://console.firebase.google.com/u/0/

### Opción A: Si VES "feeltrip_app" en la lista

```
1. En la pantalla principal, busca una tarjeta que diga "feeltrip_app"
2. Haz click en ella
3. Luego en el panel izquierdo: Build → Firestore Database
```

### Opción B: Si NO VES "feeltrip_app" (lista vacía)

```
1. Haz click en el botón: "+ Create a new project"
2. Project name: feeltrip_app
3. Continue → Continue → Create project
4. Espera 1-2 minutos
5. Cuando termine, entra al proyecto
6. Panel izquierdo: Build → Firestore Database
```

### Opción C: Si VES OTROS PROYECTOS pero no "feeltrip_app"

```
1. En la esquina arriba a la izquierda, haz click en el nombre del proyecto actual
2. Haz click en "+ Create a new project"
3. Project name: feeltrip_app
4. Continúa como en Opción B
```
