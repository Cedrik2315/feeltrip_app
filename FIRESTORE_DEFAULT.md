# ✅ SOLUCIÓN: USAR BASE DE DATOS PREDETERMINADA

## PASO 1: Crear la base de datos predeterminada (GRATIS)

```
En "Agregar base de datos":

1. Deja el nombre como "(predeterminada)" 
   → NO cambies el nombre
   → Esto NO requiere facturación

2. Ubicación: us-central1

3. Modo: Production (o Test si te lo permite)

4. Click: Crear

⚠️ IMPORTANTE: 
NO selecciones "Crear con nombre personalizado"
USA la predeterminada → SIN COSTO
```
